version 1.0.6 :

1. 自動儲存Log功能 : 
	- 儲存路徑為App路徑, Ex. .../Android/data/com.Giant.RCTestTool/files/***.txt(此路徑只有執行過App才會create)
	- 儲存連接上裝置後一直到裝置斷線之間的所有TD Log

2. 設置車架號碼功能 : 
	- 可輸入至多18碼(ASCII), 未輸入會設置為預設值EACT
	- 設置成功後需要斷線後回到裝置清單頁面點"停止掃描"再點"掃描"刷新裝置清單

3. 設置Ring功能 : 
	- 進入頁面會先讀取當前配置
	- 拖曳文字到按鈕上(按鈕不可為空), 按下儲存按鈕送出設置

4. Remote-On/Off亮度設置 : 
	- 切換白天/晚上模式會讀取當前亮度並主動開關車燈
	- 拖曳滑桿(1 ~ 100), 放開時會設置亮度

5. 指令測試 : 
	- 可觀察收到的所有callback內容
	- Clear Msg只會清空畫面上的Log, 不影響自動儲存
	- 如果為需要帶參數的指令, App給定預設值
	- 黃色字為Decode資料
	- 黑色字為Encode資料

6. 讀取車輛詳細資料 : 
	- 顯示"null"為無回應

7. 裝置清單頁面左下角顯示App版本號碼 : 
	- 點擊會顯示版本更新資訊

8. 裝置清單 : 
	- 裝置名稱|裝置address|RSSI強度
	- 只顯示Giant EBike

其他說明項目 : 
- TD指令送出延遲設置為0.5秒, 目前收到callback不會提前執行下個指令
- "功能表"按鈕只有在連上裝置後才會出現
- 連接上裝置時標題顏色會切換
